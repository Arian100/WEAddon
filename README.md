# WE Addon

A WorldEdit addon for Paper and Spigot 1.20.1 with all the features that should exist.

<p>

Can also run as a standalone.

---

### Commands

- `/blockstateplacer <block>` - A tool to place specific block states
- `/chunkreset (<chunk x>) (<chunk z>) (<world>)` - Regenerates the given chunk
- `/display <args>` - A simple way to work with block displays without external sites or nbt
- `/createexportworld <name>` - Creates an empty world to easily export schematics
- `/head <player>` - Gives you the head of a player
- `/biome <args>` - Allows you to create custom biomes and manage them without restarting the server
- `/rotate <player> <degrees>` - Allows you to rotate any player (including npc)
- `/setstone (<block>)` - Quickly create a stone block where you are
- `/world <world> (<player>)` - Allows you to quickly change worlds
